This was a programming assignment a classmate and I completed for a Data Structures and Algorithms Course that required me to implement a seam-carving algorithm for image resizing by finding and removing seams of pixels with minimum energy. I needed to create a SeamCarver class that handles energy calculations using the dual-gradient method, identifies horizontal and vertical seams, and removes them. My implementation needed to meet efficiency requirements, pass unit tests, and adhere to specified performance requirements. We wrote 295 lines of code, used a dynamic programming approach, and received a perfect score on this assignment.
